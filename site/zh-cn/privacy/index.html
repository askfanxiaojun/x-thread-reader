隐私政策

你的 Thread 内容，默认只留在浏览器里。

X Thread Reader 采用 local-first 设计。下面这页会说明插件会接触哪些数据、不会收集什么， 以及导出的 Markdown 是如何生成的。

生效日期：2026 年 4 月 2 日 适用于官网与插件

隐私概览

X Thread Reader 会读取你当前已经打开的 X/Twitter 页面中的公开 Thread 内容， 目的仅仅是帮你把这些内容整理为 Markdown，方便你个人保存、阅读或归档。

我们不提供远程解析服务，不要求注册账号，也不会在正常使用中把你导出的 Thread 内容上传到我们自己的服务器。

插件会接触哪些内容

  • 你主动触发提取时所在的 X/Twitter 当前标签页。
  • 用于生成 Markdown 的 Thread 文本内容和图片链接。
  • 只有在你点击复制时，才会调用浏览器剪贴板能力。

我们不会收集什么

  • 不会要求账号注册。
  • 不会在服务器保存你的 Thread 历史。
  • 不会基于你导出的内容做个性化分析。
  • 不会出售个人数据。

导出文件如何处理

无论是复制 Markdown 还是下载 `.md` 文件，结果都在你的浏览器会话中本地生成。 文件后续如何保存、同步或导入笔记工具，由你自己的设备和工作流决定。

官网统计

当前官网是一个静态站点。如果未来增加统计脚本，应该尽量采用隐私友好的方案， 并且不采集 Thread 导出内容。如果这部分发生变化，这页需要同步更新。

隐私政策更新

如果未来加入云端功能、账号系统，或任何服务端处理逻辑，这份隐私政策应该在功能上线前更新。

快速结论

  • 本地提取
  • 无需登录
  • 不在第三方服务器保存 Thread
  • Markdown 在客户端生成
